Tape Mark 1 - Algorithm

This diagram was published in the "Almanacco Letterario Bompiani 1962" with the description reproduced below. As far as the performative materiality of code is concerned, the phase in which the code is parsed and rendered is described as assembly.

Instructions

  1. Make combinations of 10 elements out of the 15 given without permutations and repetitions.

  2. Each element must be followed by another element whose head code has a digit corresponding to one of the two tail codes of the first element.

  3. Each element must not be followed by another element belonging to the same group.

  4. Divide the chains of 10 elements into 6 verses of 4 metric units each.

Programming

  1. Block diagram: From the study of the text and the original instructions, the diagram is derived from which the instructions for processing the electronic computer are deduced.

  2. Minutation: The instructions are transformed into symbolic language. 322 punch cards in Autocoder language are obtained.

  3. Assembly: The computer transforms the symbolic language into machine language.

  4. Preparation of magnetic tape with programme instructions and elements to operate on.
